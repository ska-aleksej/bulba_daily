# Bulba Daily 🥔

Статический дашборд для утренних встреч с картофельной тематикой. Показывает актуальные праздники, именины, погоду и дни рождения команды.

## Описание

Учебный проект — дейли-дашборд для разогрева перед встречами. Страница отображает:
- 🎉 Праздники сегодняшнего дня (через API my-calend.ru)
- 🎂 Дни рождения команды (виджет в хедере + выпадающий список)
- 👼 Именины (кто отмечает день ангела)
- 🌡️ Погода в разных городах (Open-Meteo API)
- ⚙️ Настройки через localStorage (название команды, кастомные дни рождения)

## Технологии

- HTML5
- CSS3 (911 строк)
- Vanilla JavaScript (ES6 modules)
- Open-Meteo API для погоды
- my-calend.ru API для праздников
- Никаких фреймворков и сборщиков

## Функционал

### Виджет дней рождения
- Показывает ближайший день рождения команды
- Выпадающий список со всеми днями рождения
- Автоматическое определение дней рождения в списке праздников
- Цветовая индикация: сегодня, скоро (3-7 дней), обычно

### Погода
- Отображение погоды в городах команды (Познань, Минск, Омск, Самара)
- Использует Open-Meteo API (бесплатный, без API ключа)
- Показывает температуру, ощущаемую температуру, ветер и влажность
- Фоновые изображения для каждого города

### Настройки
Настройки сохраняются в localStorage браузера:
- `bulba.daily.team.name` — название команды (заголовок страницы)
- `bulba.daily.custom.birthdays` — кастомные дни рождения

**Пример настройки названия команды:**
```javascript
window.setSetting('team.name', 'My Awesome Team');
```

**Пример добавления дня рождения:**
```javascript
window.setSetting('custom.birthdays', [
  { name: 'Иван', day: 15, month: 0 },  // 15 января
  { name: 'Мария', day: 23, month: 2 }   // 23 февраля
]);
```

## Как использовать

1. Клонировать репозиторий
2. Открыть `index.html` в браузере
3. Наслаждаться перед утренней встречей ☕

## Запуск

Просто откройте `index.html` в любом современном браузере. Никаких дополнительных установок не требуется.

## Внешние API

- **Open-Meteo** — бесплатный API погоды (не требует API ключа)
- **my-calend.ru** — праздники и именины (через proxy api.codetabs.com для обхода CORS)
